发表于 2017-11-06

创建RegExp对象，两种方式：

  • 构造函数方式
    var reg=new RegExp(“表达式”,”附加参数”);

  • 普通方式

    var reg=/表达式/附加参数

    在第一种参数中，正则表达式一般以javascript字符串的形式存在，需要使用单引号或双引号引起来。在第二种方式中，则嵌套在“/”字符之间，不能使用引号，两种方式中，附加参数是可选的，用来扩展正则表达式的含义，主要有三个参数：

    g:全局匹配。如果设置了这个参数，使用该正则表达式模式对
    i:忽略大小写
    m:多行匹配

正则表达式元字符

1.限定符

限定符用于指定前面的字符或者组合项连续出现的次数

限定符 描述
n+ 匹配任何包含至少一个 n 的字符串。
n* 匹配任何包含零个或多个 n 的字符串。
n? 匹配任何包含零个或一个 n 的字符串。
n{X} 匹配包含 X 个 n 的序列的字符串。
n{X,Y} 匹配包含 X 至 Y 个 n 的序列的字符串。
n{X,} 匹配包含至少 X 个 n 的序列的字符串。
n$ 匹配任何结尾为 n 的字符串。
^n 匹配任何开头为 n 的字符串。
?=n 匹配任何其后紧接指定字符串 n 的字符串。
?!n 匹配任何其后没有紧接指定字符串 n 的字符串。
2.选择匹配符

选择匹配符只有一个，即“|”字符，用于匹配两个中间的一个。

3.字符匹配符
符号 描述
[…] 匹配方括号字符集中任意一个字符，如[abc],字符“a”,”b”,”c”都可以匹配
[^…] 匹配方括号中未包含的字符
[a-z],[1-9] 指定范围内的任意字符，加一个“^”号用于匹配不在范围内的字符，方法同上
\d 匹配任意一个数字字符，等效于[0-9]
\D 匹配任意一个非数字字符
\s 匹配任意空白字符，包括空格，制表符等
\S 匹配任意非空白字符
\w 匹配任何英文字母，数字和下划线
\W 匹配任何非英文字母和数字类字符，单不包括下划线，
. 匹配除换行符\n之外的任何单个字符
